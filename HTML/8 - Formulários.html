Forms

Takes input from the user

"<"form
action= "pagina pra onde vai ser enviado"
method= "post" ou "get"
(post: encripta e envia, get: só envia) ">"
"<"/form>

tipos de input

input:
coleta dados informados pelo usuário
type: tipo de input
name: variável que armazena o que o usuário vai inserir placeholder: o que aparece na caixinha
id: a label (ver abaixo)

labels

identifica o input para melhor organização e acessibilidade

< label
for= variavel que identifica, deve ser única para cada input (será a id no input)
>
label que o usuário vai ver
<label>
